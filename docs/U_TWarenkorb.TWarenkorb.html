Beschreibung

Felder

Content: TIntArray;

Beinhaltet alle ausgewählten Fälle bzw. deren IDs (0-12).

Functions: array[0..12] of TRVGFunc;

Beinhaltet die Referenzen zu den TRVGFunc-Funktionen, zur Berechnung der Fälle.

intern const Names: array[0..12] of string = ( ('Brief'), ('Brief-Mahn'), ('Brief-Mahn-Klage-Gericht-Verlieren'), ('Brief-Mahn-Klage-Vergleich'), ('Brief-Mahn-Klage-Berufung'), ('Brief-Mahn-Vollbesch'), ('Brief-Mahn-VollBesch-Voll'), ('Brief-Mahn-VollBesch-Klage-Gericht-Verlieren'), ('Brief-Mahn-VollBesch-Klage-Vergleich'), ('Brief-Mahn-VollBesch-Klage-Berufung'), ('Klage-Gericht-Verlieren'), ('Klage-Vergleich'), ('Klage-Berufung') );

Die Namen aller möglichen Fälle. Je nach ID (0-12) wird der Name des dazugehörigen Falls im Array gespeichert, um im TGrid angezeigt zu werden.

Methoden

procedure Add(itemId: integer; aGrid: TGrid);

Vergrößert das TGrid um eine Zeile und übergibt die ID des ausgewählten Falls, damit später der Name und Preis des richtigen Falls eingetragen wird.

procedure Update(aGrid: TGrid);

Aktualisiert das TGrid, damit alle neu ausgewählten Fälle angezeigt werden.

function getPrice(itemId: integer; streitwert: real):real;

Ruft die durch den Constructor Create gespeicherte Funktion auf.

Parameter
itemID
Die durch Knopfdruck übergebene ID des ausgewählten Falls.
streitwert
Der Streitwert des Falls, der an die Berechnungsfunktionen weitergegeben wird.
Result

Den ausgegeben Preis auf zwei Nachkommastellen gerundet.

constructor Create;

Speichert die ausgewählte Berechnungsfunktion in Functions, die später ausgeführt werden soll.
