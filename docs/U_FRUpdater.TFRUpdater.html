Deklaration

type TFRUpdater = class(TObject)

Beschreibung

Ruft den aktuellen Leitzins ab und liest ihn aus einer lokaler Datei.

Diese Klasse ermöglicht das Abrufen des aktuellen EZB-Leitzinses aus dem Internet über die API des Statistical Data Warehouse der EZB. Außerdem enthält sie einige Funktionen, die bei der Abfrage helfen.

procedure updateFR;

Ruft den den aktuellen EZB-Leitzins aus dem Internet äber die API des Statistical Data Warehouse der EZB ab. Dieser wird in die durch FFileName spezifizierte Datei als Float (z.B. 0,000) abgespeichert.

function readFR: real;

Liest den EZB-Leitzins aus der in Create angegebenen Datei aus. Die Datei muss sich im aktuellen Verzeichnis befinden.

Result

Der Leitzins als Float.

procedure writeFRToFile(FR: real);

Speichert den übergebenen Leitzins in die durch FFileName spezifizierte Datei als Float (z.B. 0,000).

Parameter
FR
Der Leitzins als Float.
function HasInternet: boolean;

Überprüft ob Verbindung zu FURL besteht.

Result

function FileExist: boolean;

Überprüft ob die Datei an FFileName existiert.

Result

Falls sie existiert: True, ansonsten False.
