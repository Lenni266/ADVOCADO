Unit

U_FRUpdater

Deklaration

type TFRUpdater = class(TObject)

Beschreibung

Ruft den aktuellen Leitzins ab und liest ihn aus einer lokaler Datei.

Diese Klasse ermöglicht das Abrufen des aktuellen EZB-Leitzinses aus dem Internet über die API des Statistical Data Warehouse der EZB. Außerdem enthält sie einige Funktionen, die bei der Abfrage helfen.

Felder

In der Klasse abgespeicherte URL.

FFileName: string;

Absoluter Absoluter Pfad zur Datei, welche den Leitzins enthält. Wird generiert durch den in Create angegebenen Dateinamen und das aktuelle Verzeichnis.

Methoden

function GetPage(aURL: string): string;

Gibt eine Internetseite als String aus.

Parameter
aURL
Die URL der Seite als String.
Result

Der Inhalt der Seite als String.

procedure updateFR;

Ruft den den aktuellen EZB-Leitzins aus dem Internet über die API des Statistical Data Warehouse der EZB ab. Dieser wird in die durch FFileName spezifizierte Datei als Float (z.B. 0,000) abgespeichert.

function readFR: real;

Liest den EZB-Leitzins aus der in Create angegebenen Datei aus. Die Datei muss sich im aktuellen Verzeichnis befinden.

Result

Der Leitzins als Float.

function HasInternet: boolean;

Überprüft ob Verbindung zu FURL besteht.

Result

Falls Verbindung besteht: True, ansonsten False.

function FileExist: boolean;

Überprüft ob die Datei an FFileName existiert.

Result

Falls sie existiert: True, ansonsten False.

constructor Create(URL, FileName: string);

Constructor der Klasse.

Parameter
URL
URL der Anfrage an die API als String.
FileName
Name der zu erstellenden bzw. schon existierenden Datei als String. Lediglich der Name, kein absoluter/relativer Pfad.
